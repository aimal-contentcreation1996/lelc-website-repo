About this course

“DATA” has many definitions, the context of which can change according to functional uses. This is a basic programme providing fundamentals of data analysis at the end of which you will be able to read data from sources like CSV, BigQuery, SQL, and how to use libraries like NumPy and Pandas, and visualize data.
